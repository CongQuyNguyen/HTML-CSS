Thẻ code

Thẻ code là một thẻ có ngữ nghĩa. Cũng giống như thẻ <h1> hoặc các thẻ h khác, ngoài ra còn có thẻ <em> và thẻ <strong>.
Khác với các thẻ này thì các thẻ định dạng font như <b>, <u>, <i> là các thẻ không có nghĩa.

Ta sẽ dùng thẻ <pre> và thẻ <code>

Dưới đây là đoạn code định dạng của một file html

<!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=p, initial-scale=1.0" />
        <title>Document</title>
      </head>
      <body>
        <p>Chúng ta sử dụng thẻ &lt;h1&gt; để làm tiêu đề:</p>
        <p>&lt;h1&gt;Đặt tiêu đề vào đây&lt;/h1&gt;</p>
        <br />
        <p>Một số ký tự đặt biệt:</p>
        &copy;: Đây là mã copyright
    
        <h2>Ta có thể lên trang web để tìm kiếm các mã, kể cả icon</h2>
      </body>
    </html>
